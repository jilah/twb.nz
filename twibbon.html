Inspirasi dunia sekitar Anda. Setiap hari, di seluruh dunia, jutaan Bingkai Profil digunakan untuk merayakan momen-momen penting, memeriahkan kemenangan tim, dan menunjukkan dukungan pada gerakan-gerakan tertentu.

Kelola Bingkai

Tahun lalu, kami membatasi kemampuan membuat bingkai profil disini hanya untuk organisasi yang berwenang.

Kami kini melanjutkan upaya tersebut, sehingga Bingkai profil dari Halaman dan profil yang tidak disetujui tidak bisa lagi diterapkan ke foto profil baru.

Kini, hanya bingkai profil dari layanan pemerintah atau organisasi tertentu yang memberikan informasi terkait wabah COVID-19 yang akan tersedia.

Perubahan ini mencerminkan fokus kami yang berkelanjutan untuk membantu orang mengungkapkan dukungan mereka terhadap isu penting, seperti pemungutan suara dan informasi kesehatan yang bisa diandalkan.

Tambahkan tampilan yang disesuaikan ke profil Facebook hanya dalam beberapa langkah. Buat bingkai di Frame Studio untuk teman dan pengikut Anda menggunakan karya seni original Anda sendiri atau pilih dari template kami.

Cara Membuat Bingkai

  • Akses Twibbon Generator.
  • index.md : Website page (for now, this page).
  • _includes/head.html : File to add custom code to <head> section.
  • _includes/scripts.html : File to add custom code before </body>. You can change footer at here.
  • _sass folder : Related scss files can be found at this folder.
  • css/main.csss : Main scss file.
  • README.md : A simple readme file.
